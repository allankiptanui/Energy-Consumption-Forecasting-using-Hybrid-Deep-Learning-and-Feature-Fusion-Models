# energy-forecasting-hybrid

Hybrid energy consumption forecasting repo

Recommended workflow:
1. Place raw CSV(s) into `data/raw/` (e.g., `energy_data.csv`, optional weather CSV).
2. Adjust `config/config.yaml`.
3. Run `python experiments/run_all.py` (or use notebooks/exploration.ipynb).

See `requirements.txt` for packages.
